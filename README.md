# Unscrollable

React:

```jsx
import Unscrollable from 'unscrollable'

const App = () => (
  <Unscrollable>
    Scrolling is limited to take effect within the component
  </Unscrollable>
)
```

Vue:

```vue
<script setup lang="ts">
import Unscrollable from 'unscrollable/vue'
</script>

<template>
  <Unscrollable>
    Scrolling is limited to take effect within the component
  </Unscrollable>
</template>
```
